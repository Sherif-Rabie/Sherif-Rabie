Modelling

#splitting the data for cross validation
set.seed(88)
split = sample.split(HR$left, SplitRatio = 0.75)
HRTrain = subset(HR, split == TRUE)
HRTest = subset(HR, split == FALSE)

Building a Logitics Regression model. \(Y = X\beta + \epsilon\)

#building a logistic model
model = glm(left ~ satisfaction_level + time_spend_company + last_evaluation + average_montly_hours + salary , data=HRTrain, family=binomial)
#checking the odds
